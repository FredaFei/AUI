# Amazing-react

一套PC&H5均适用的组件库，基于react hooks + typeScript 开发，无额外依赖。

### 快速上手

**安装**

```
  $ yarn add amazing-ui-react
  $ npm install amazing-ui-react
```

**使用组件**

```jsx harmony

import { Button } from "amazing-ui-react";
import "amazing-ui-react/dist/lib/index.css";

<Button>submit</Button>

```
