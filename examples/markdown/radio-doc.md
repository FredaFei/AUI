### API

| 参数         |说明        |类型 |默认值 |可选值 |
| ------------ | :----------------|:------- | :----- | :----- |
| checked |设置选中项的值 |Boolean |false |true |
| disabled      |设置按钮失效状态  |Boolean |false |true |
| className      |自定义类名  |string |—— |—— |
| style      |自定义样式  |React.CSSProperties |—— |—— |
| onChange    | 选中项改变时触发 |       (event) => void |—— |—— |
