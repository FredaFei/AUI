### 注意

该组件仅适用于移动端，组件交互请切换到移动端模式查看。

### API

| 参数         |说明        |类型 |默认值 |可选值 |
| ------------ | :----------------|:------- | :----- | :----- |
| pullingHeight      |设置上下拉动距离|Number |100 |—— |
| className      |自定义类名  |String |—— |—— |
| style      |自定义样式  |React.CSSProperties |—— |—— |
| onPullUp    | 下拉时触发 |       () => void |—— |—— |
| onPullDown    | 上拉时触发 |       () => void |—— |—— |
