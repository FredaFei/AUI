### API

| 参数         |说明        |类型 |默认值 |可选值 |
| ------------ | :----------------|:------- | :----- | :----- |
| icon         |设置按钮的图标类型 |String |—— |—— |
| iconPosition |设置图标的摆放位置 |String |left |right |
| disabled      |设置按钮失效状态  |Boolean |false |true |
| className      |自定义类名  |String |—— |—— |
| style      |自定义样式  |React.CSSProperties |—— |—— |
| onClick    | 点击按钮时触发 |       (event) => void |—— |—— |
