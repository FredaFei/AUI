### API

| 参数         |说明        |类型 |默认值 |可选值 |
| ------------ | :----------------|:------- | :----- | :----- |
| display         |当前展示日期 |Date/String |—— |—— |
| firstDayOfWeek |设置一周的起始 |Number |1 |0-6|
| className      |自定义类名  |String |—— |—— |
| style      |自定义样式  |React.CSSProperties |—— |—— |
| extraFooter    | 添加额外的页脚 | () => ReactNode |—— |—— |
| onChange      |时间发生变化的回调  |(date: Date/String) => void |—— |—— |
