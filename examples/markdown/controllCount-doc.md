### API
### Collapse

| 参数         |说明        |类型 |默认值 |可选值 |
| ------------ | :----------------|:------- | :----- | :----- |
| count         |当前数量 |Number |1 |—— |
| maxCount |最大数量 |Number|10 |—— |
| minCount |最小数量|Number |1 |—— |
| readOnly |输入框是否只读|Boolean |false|`true` |
| className      |自定义类名  |String |—— |—— |
| style      |自定义样式  |React.CSSProperties |—— |—— |
| onChange    | 数量改变时回调 |       (count) => void |—— |—— |
