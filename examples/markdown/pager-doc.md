### API

| 参数         |说明        |类型 |默认值 |可选值 |
| ------------ | :----------------|:------- | :----- | :----- |
| current         |设置当前页数 |Number |—— |—— |
| defaultCurrent |设置默认的当前页数 |Number |—— |—— |
| total |数据总数 |Number |—— |—— |
| visibleQuickJumper      |是否可以快速跳转至某页  |Boolean |false |true |
| className      |自定义类名  |string |—— |—— |
| style      |自定义样式  |React.CSSProperties |—— |—— |
| onChange    | 页码改变的回调，参数是改变后的页码 |       (page) => void |—— |—— |
